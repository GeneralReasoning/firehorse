# Contributing to Firehorse

Thanks for your interest in contributing to Firehorse!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/firehorse.git`
3. Install in development mode: `pip install -e ".[react,resum]"`
4. Create a branch for your changes: `git checkout -b my-feature`

## Development

Run tests:
```bash
pytest
```

## Submitting Changes

1. Ensure your code follows the existing style
2. Add tests for new functionality
3. Update documentation if needed
4. Submit a pull request

## Reporting Issues

Please open an issue on GitHub with:
- A clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Environment details (Python version, OS, etc.)
